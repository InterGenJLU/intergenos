# LFS Version Pinning Decision — LFS 13.0

**Date:** March 31, 2026
**Decision:** Pin InterGenOS base build to LFS 13.0-systemd

## LFS 13.0 Key Facts

- **Released:** March 5, 2026
- **First systemd-only release** (SysVinit variant discontinued)

## Why 13.0 Over 12.3

| | LFS 12.3 (March 2025) | LFS 13.0 (March 2026) |
|---|---|---|
| Kernel | 6.13.2 | 6.18.10 |
| GCC | 14.2.0 | 15.2.0 |
| glibc | 2.41 | 2.43 |
| systemd | ~255 | 259.1 |
| Binutils | 2.44 | 2.46.0 |
| Init system | systemd or SysVinit | systemd only |

- 13.0 is current stable (released 25 days before this decision)
- systemd-only aligns with InterGenOS requirements (systemd-homed for DE integration)
- Modern toolchain versions reduce compatibility issues
- 36 packages updated since 12.4, including security fixes

## Known Security Fixes in 13.0

- expat, glibc, openssl, Python, vim, zlib all received security updates
