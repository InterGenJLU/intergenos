# InterGenOS

**A Linux distribution built from source with switchable desktop environments and a tiered local AI assistant.**

InterGenOS is a passion project — a Linux From Scratch distribution designed to put the user in control of their own machine. Every package is compiled from source with deliberate choices. The built-in AI assistant helps you learn Linux, not hide it from you.

## Features (In Development)

- **Built from source** — LFS 12.3 base, every component chosen deliberately
- **Switchable desktop environments** — pick your DE at login with isolated configs
- **Local AI assistant** — hardware-detected tiers, fully offline, no cloud dependency
- **Wayland-native** — GNOME, KDE Plasma, COSMIC

## Status

Revival in progress. Originally built in 2015-2016 (build_001 through build_003). Now being rebuilt with modern tooling, AI integration, and a decade more experience.
